# Open Openings

MVP веб-приложения для изучения и тренировки шахматных дебютов.
Frontend-only. Интерфейс только на английском языке.

Доступные дебюты хранятся прямо в этом репозитории в специальном формате.
Формат описан в файле `src/entities/openings/types.ts`.
Файл `src/entities/openings/index.ts` экспортирует все доступные дебюты.

## Приоритеты продукта

- UX: простые сценарии, понятные тексты/ошибки, доступность.
- Надёжность, предсказуемое поведение.
- Качество кода, читаемость.

## Полезная информация

Все файлы в директории `/docs` обязательны к прочтению.

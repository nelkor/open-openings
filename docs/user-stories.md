# User Stories

## Пользователь хочет изучить определённую вариацию дебюта

- Открыл главную страницу.
- Выбрал дебют.
- Выбрал вариацию.
- Выбрал режим изучения.
- Перешёл на /learn.
- Делает ходы по подсказкам, читает теорию.
- Прочитал поздравление о пройденном уроке.
- Перешёл на главную страницу.

Пользователь мог выбрать не только вариацию, но и начальную линию дебюта.

Также после прохождения урока есть возможность повторить урок сначала,
либо перейти в режим тренировки. В этом случае пользователь остаётся
на доске.

## Пользователь хочет тренировать определённую вариацию дебюта

- Открыл главную страницу.
- Выбрал дебют.
- Выбрал вариацию.
- Выбрал режим тренировки.
- Перешёл на /practice.
- Играет против бота, который идёт по линии дебюта.
- Ошибся, игра остановилась.
- Посмотрел на ошибку, прочитал теорию.
- Начал тренировку заново.

После прохождения любой тренировки можно вернуться на главную страницу,
начать заново или перейти в режим изучения текущей линии.

## Пользователь хочет потренировать дебют целиком

- Открыл главную страницу.
- Выбрал дебют.
- Выбрал режим тренировки.
- Перешёл на /practice.
- Играет против бота, который идёт по случайной линии.
- Доиграл линию правильно до конца.
- Начал заново (бот выбрал другую случайную линию).

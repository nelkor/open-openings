# Tech Stack

Используем TypeScript + Vue3 + Vue Router. В компонентах используем только
`<script setup lang="ts">`. Стили пишем в SCSS, без UI-библиотек.

Не используем библиотеки состояний, такие как Pinia. Если нужно хранить
состояние вне компонента, объявляем `ref` в `.ts`-файле.

Не используем автоматическое тестирование.

Шахматную доску реализуем библиотекой `vue3-chessboard`.
